  <name>Task 1: Install and Configure Vitest Framework</name>
  <files>vitest.config.mts, package.json</files>
  <action>
Install Vitest 4.1 with React Testing Library and create configuration (per VALIDATION.md Wave 0 requirements).

1. Install Vitest and testing dependencies:
   ```bash
   pnpm add -D vitest@4.1 @vitejs/plugin-react jsdom @testing-library/react @testing-library/dom vite-tsconfig-paths
   ```

2. Create vitest.config.mts in project root:
   ```typescript
   import { defineConfig } from 'vitest/config'
   import react from '@vitejs/plugin-react'
   import tsconfigPaths from 'vite-tsconfig-paths'

   export default defineConfig({
     plugins: [react(), tsconfigPaths()],
     test: {
       environment: 'jsdom',
       globals: true,
       setupFiles: []
     }
   })
   ```

3. Add test scripts to package.json:
   ```json
   {
     "scripts": {
       "test": "vitest",
       "test:run": "vitest --run"
     }
   }
   ```

Note: This creates the test framework infrastructure. Test files will be created in Task 2.
  </action>

  </verify>
